HJC Depend - Must read for this task

HJC depend is an eclipse plug-in that helps you discover bugs in your source code. To run it, you will open the HTML file in your eclipse editor you want to find bugs in and then click on the "Run" button.

What does it do?

When you hit run, the plug-in will first read your HTML file and will locate the JavaScript and CSS files that are linked to the HTML file using the link and script tags.

It will then present you a list of errors and warnings that may be bugs in the opened HTML file and the linked JavaScript and CSS files.
