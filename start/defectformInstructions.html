Defects reporting Instructions


You are required to fill this defect reporting google survey form for every defect you find:-

Please enter your unique identifier

The unique identifier that we generated for you will go here.

Group A or B?

The group you have been allotted to.

Please describe the bug in one or two lines

Sample:- The directory path "../some/random/dir.js" is incorrect. The page is unable to load required JavaScript.

Please enter the time (mm:ss) from the stop watch it took you to find this bug. (minutes:seconds)

Sample:- 02:30
The above time would be read as 2 minutes and 30 seconds.

Please enter the file name(s) where you found the bug.

Sample:- index.html
The filename where the defect is present, you could add comma separated filenames if you think that the defect is spread in multiple files.

Please enter the Line Number(s) where the bug is located.

Sample:-56
The line number where the defect is present, you could add comma separated line number if you think that the defect is spread in multiple lines.
