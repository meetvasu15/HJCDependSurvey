Group B

You are given a USB flash drive, please log in to any one of the desktops and insert this usb drive. When you open the drive, you will see two folders:-

1. eclipse
2. workspace



  • Please go inside the eclipse folder and launch(double click) the file eclipse.exe.
  • You will be prompted to select a workspace by eclipse. Please select the "workspace" folder located inside the USB Flash drive.
  • When eclipse launches you will see that we have set up a project for you please open it (if you see the eclipse welcome screen, please close that screen and you should see the project screen).
  • Now you can click the source tree under when you open the project folder to view source files.
  • For this exercise we ask you to use eclipse plugin (HJCDepend) to assist you resolve the defect list you will be presented shortly.
  • To open the plugin please choose "Window" from menu bar and then in "show view" choose "other" and then type "hjcdepend" in the search bar. When you click on the "HJCDepend" option, you should see a new window near your console that has title "HJC depend" and has a "Run" button.
  • To use this tool you should open the html file in your editor and then press run, you should then see a list of defects the tool recognized or if it does not display anything, then probably you don't have any more bugs.
