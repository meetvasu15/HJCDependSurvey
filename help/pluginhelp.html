Understanding HJC Depend Messages



Sample error/warning Reported Description
Please open the HTML file you want to analyze and then hit 'Run HJC Depend'. To run the analysis on your code, we read the HTML file as a point of entry. Therefore, you need to open the html file in eclipse editor and while staying on it, hit 'Run HJCDepend'
Javascript tried to fetch unavailable HTML reference 'someHTMLId' In one of your JavaScript files, there was a reference to an id which is not present in the HTML file.
Eg document.getElementById('someHTMLId');
Access to undefined DOM element, HTML reference 'someHTMLId' unavailable. In one of your JavaScript files, there was a reference to an id which is not present in the HTML file and then the code tried to write to that unavailable HTML element.
Eg: document.getElementById('someHTMLId').innerHTML = 'Something odd';
Access to an undefined DOM element 'domElt', check initialization of this variable This is similar to the one above, the difference is that JavaScript tries to write to an assigned variable
Eg: var domElt = document.getElementById('someHTMLId');
domElt.innerHTML="content";
Could not find css class 'myCssClass' in any included CSS file. This is a warning, HJC depend did not find any matching class applied to HTML in css files
Call to undefined function 'unavailableFunction' HJC Depend could not find any function with name 'unavailableFunction' in all your JavaScript files.
Invalid argument count of function 'functionName' in HTML file This is a warning, your HTML called a JavaScript function with invalid number of arguments.
Eg: your HTML is: <\div onclick=\"someFunction('param1')\"\>
and your JS declares this function as someFunction('param1', 'param2')
